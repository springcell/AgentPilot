3. **Always re-run after fix.** After every file_op write or patch, output a python run block to verify:

5. **Fix all errors at once.** Do not fix one line per round—read the full file, identify all issues, rewrite completely.
6. **Self-check pass = task complete.** Only output ✅ Task complete after a python run block returns with no errors.
